# E commerce Sales Dashboard Project Excel
Worked on creating a dynamic dashboard to analyze e-commerce sales performance, customer behavior, and product trends with meaningful business insights and professional visualization.

✨ Key Highlights:
✔️ Total Orders, Quantity & Revenue Tracking
✔️ Average Customer Rating & Delivery Time Analysis
✔️ Last 13 Weeks Sales & Quantity Trends
✔️ Gender-wise Customer Breakdown
✔️ Platform-wise Purchase Analysis (Website, App, Instagram, Partner Apps, Target.com)
✔️ Product Popularity Analysis
✔️ Customer Buying Pattern Insights
✔️ Region-wise Customer Distribution
✔️ Delivery Performance Monitoring
✔️ Interactive Filters & Slicers for Dynamic Exploration

🛠️ Tools Used:
Microsoft Excel
Pivot Tables & Pivot Charts
Data Visualization Techniques
Interactive Slicers
Dashboard Design & Formatting
